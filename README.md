**Open Custom GPT** provides a user-friendly solution to quickly setup a custom GPT and add to your site.

### Stack

- Next.js
- OpenAI
- Tailwind

### Run the project locally

Minimum requirements to run the projects locally

- Node.js v18
- OpenAI API Key

```shell
npm install

npm run dev

# visit http://localhost:3000
```
